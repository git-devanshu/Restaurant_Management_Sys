=== HTTP error codes / Rest API error codes ===

201 : created
202 : accepted
203 : non authoritative information
204 : no content

400 : bad request
401 : unauthorized
403 : forbidden
404 : not found
409 : conflict

500 : internal server error 
501 : not implemented
502 : bad gateway
503 : service unavailable
504 : gateway timeout
599 : network timeout
